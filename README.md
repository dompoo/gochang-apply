# 🥁 고창농악 전수 빠른 신청 스크립트

# 사용법

## 1단계: 사전 준비 (신청 전날까지)

### 1-1. 스크립트 복사

[`gochang_apply.js`](./gochang_apply.js) 파일의 전체 내용을 복사합니다.

### 1-2. 수정한 스크립트 저장

메모장, 메모 앱 등 아무 곳에나 **수정 완료된 스크립트 전체**를 저장해둡니다.

### 1-3. 내 정보 입력

상단 `CONFIG` 부분을 본인 정보로 수정합니다.

```js
const CONFIG = {
  name: '홍길동',              // 신청자 이름
  tel: '01012345678',        // 연락처 (하이픈 없이 숫자만)
  depositor: '홍길동',         // 입금자명 (본인과 다를 수 있음)
  headcount: 1,              // 총 인원수
  affiliation: '개인',        // 소속 (개인 / 단체명)
  affiliationDetail: '개인',  // 소속 상세

  participants: [
    { name: '홍길동', instrument: '장구', gender: '남' },
  ],

  autoSubmit: false,         // true로 바꾸면 자동 제출 (주의!)
};
```

**악기(`instrument`) 종류**

| 값 | 악기 |
|---|---|
| `'쇠'` | 꽹과리 |
| `'장구'` | 장구 |
| `'북'` | 북 |
| `'소고'` | 소고 |
| `'태평소'` | 태평소 |

### 1-4. 🚨 권한 열기 : allow pasting

처음 사용하는 경우, 콘솔에 붙여넣기가 막혀있을 수 있습니다.
미리 콘솔에 붙여넣기를 시도해보시고, 안된다면 콘솔에 `allow pasting` 입력하고 엔터를 눌러서 권한을 열어주세요.

## 2단계: 신청 방법 (신청 오픈 시)

### 2-1. 신청 페이지 접속

신청이 오픈되면 **즉시** 신청 페이지에 접속합니다.

### 2-2. 개발자 도구 열기

키보드에서 `F12`를 눌러서 개발자 도구를 엽니다.

### 2-3. Console 탭 확인

개발자 도구가 열리면 상단에서 **Console** 탭을 클릭합니다.

### 2-4. 스크립트 붙여넣기 + 실행

1. 콘솔 입력창에 **스크립트를 붙여넣기**
2. 엔터를 눌러 실행

### 2-5. 확인 및 제출

- 폼이 자동으로 채워진 것을 **눈으로 확인**합니다
- 이상이 없으면 **신청 버튼을 직접 클릭**하여 제출합니다

`autoSubmit: true`로 설정한 경우, 붙여넣기 + Enter만으로 바로 제출됩니다. 빠르지만, 실수할 경우 되돌릴 수 없으니 신중하게 사용하세요!

## 여러명 신청하는 경우

`participants` 배열에 참가자를 추가하고, `headcount`를 추가하세요.

```js
headcount: 3,

participants: [
  { name: '홍길동', instrument: '장구', gender: '남' },
  { name: '김영희', instrument: '북', gender: '여' },
  { name: '이철수', instrument: '쇠', gender: '남' },
],
```
